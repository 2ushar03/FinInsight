# FinInsight

## AI-Powered Research Tool for Banking

FinInsight is an advanced research tool designed to assist banking professionals in making informed decisions by analyzing user-specified URLs. Leveraging cutting-edge technologies like Generative AI, LangChain, Hugging Face, Faiss, and React, FinInsight extracts and processes relevant banking data efficiently.

### Features

- **URL Analysis**: Analyze user-specified URLs to extract pertinent banking information.
- **Enhanced Decision-Making**: Provide insights that aid in more informed decision-making processes.
- **Text Chunking and LLMs**: Implement text chunking techniques and utilize large language models (LLMs) to create embeddings, enhancing information retrieval and context understanding.
- **Efficient Data Storage**: Utilize Faiss for efficient data storage and retrieval, ensuring quick access to relevant information.
- **Retrieval-Augmented Generation (RAG)**: Employ RAG techniques to provide accurate answers along with source citations, improving the reliability of the information.

### Technologies Used

- **Generative AI**
- **LangChain**
- **Hugging Face**
- **Faiss**
- **React**
